# ALAS Browser Policy Site

This repository hosts the public policy website for ALAS Browser.

## Deploy target
This site is configured for **GitHub Pages**.

## Live URL
`https://alaslife.github.io/alas-browser-policy/`

## Site files
- `public/index.html` - Privacy Policy and Terms page
- `public/.nojekyll` - ensures static passthrough
- `.github/workflows/pages.yml` - Auto deploy workflow for GitHub Pages

## Deploy steps (GitHub Pages)
1. Open repo: `alaslife/alas-browser-policy`.
2. Go to **Settings > Pages**.
3. Set **Source** to **GitHub Actions**.
4. Push to `main`.
5. Wait for workflow `Deploy static site to GitHub Pages` to succeed.
